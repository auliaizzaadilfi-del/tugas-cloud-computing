Selamat datang di website saya. Halaman ini adalah bukti nyata implementasi teknologi Cloud Computing menggunakan model Platform as a Service (PaaS).

Bagaimana Website Ini Bekerja? ☁️

📁 Cloud Repository (GitHub)

Semua kode sumber (HTML, CSS, JS) dari website ini disimpan secara aman di cloud menggunakan GitHub. Ini berfungsi sebagai pusat penyimpanan dan manajemen versi kode.

🚀 Cloud Deployment (Vercel)

Saya menggunakan Vercel (PaaS) yang terhubung langsung dengan GitHub. Saat ada perubahan kode, Vercel secara otomatis membangun dan meluncurkan website ini ke internet (CI/CD).

⚡ Jaringan Global (CDN)

Website ini tidak berada di satu server fisik saja. File website didistribusikan ke berbagai server di seluruh dunia menggunakan Content Delivery Network agar bisa diakses dengan sangat cepat.
